# Interview Tracker

Interview Tracker es una aplicación sencilla hecha con React + Vite para llevar el control de tus candidaturas de empleo.

## ¿Qué hace?

- Muestra una lista de candidaturas de ejemplo.
- Permite buscar por empresa o puesto.
- Permite filtrar por estado.
- Permite mostrar u ocultar un formulario para añadir nuevas candidaturas.
- Cada candidatura muestra información básica como empresa, fecha, notas y estado.

## Tecnologías

- React
- Vite
- JavaScript
- CSS modular por componentes

## Estructura del proyecto

- `src/components` → componentes de la interfaz
- `src/data` → datos de ejemplo de candidaturas
- `src/App.jsx` → componente principal
- `src/index.css` → estilos base

## Ejecutar localmente

1. Instala dependencias:
   ```bash
   npm install
   ```
2. Inicia el proyecto:
   ```bash
   npm run dev
   ```
3. Abre la URL que aparezca en la consola, normalmente:
   ```bash
   http://localhost:5173/
   ```
